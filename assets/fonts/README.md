# Video font

`NotoSansSC-VF.otf` is the Simplified Chinese subset variable font from the
official [Noto CJK](https://github.com/notofonts/noto-cjk) repository:

- source: `Sans/Variable/OTF/Subset/NotoSansSC-VF.otf`
- license: SIL Open Font License 1.1, included as `OFL.txt`
- SHA-256: `d13ed01ec8aa45d6178999b648e96fb92150683e9f8e2a581f2acf208dcbe44b`

The video typography theme uses the variable `wght` axis for Regular (400),
Medium (600), and Bold (700). Keeping the font in the project makes offline
4K exports independent from host font configuration.
