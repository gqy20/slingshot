# T002 · 更重的物体真的飞得更远吗？

## 文档状态

- 状态：候选 Episode，依赖 T001 空气阻力求解器
- 建议时长：5 分钟
- 主要受众：理工科大学生、研究生、工程师、科学爱好者
- 主要渲染：Godot
- 可复用基础：T001 轨迹、阻力、角度与结果曲线组件
- 新增能力：实验条件切换、同物体跨实验颜色语义、二维参数扫描

## 一句话承诺

用同一对轻球和重球做三次看似相同的发射，展示“谁飞得远”并不先由质量决定，而是先由“怎样保证公平”决定。

## 观众认知转折

```text
进入前：重物惯性大，所以更能顶住空气阻力，应该飞得更远。
第一层：真空中同速发射，质量根本不改变轨迹。
第二层：有空气且同速发射，重物受到的阻力减速度较小。
第三层：固定发射能量时，重物的初速度更低，比较结果重新变化。
离开时：没有控制条件，“更重是否飞得更远”不是一个完整问题。
```

## 对索引初稿的修正

在理想弹簧模型且效率固定时：

```text
E_k = η · 1/2 kx²
```

“固定初始动能”和“使用同一根弹簧、相同拉伸距离”是同一类条件，不应被包装成两个独立实验。正式视频将其合并，并把这种等价关系作为建模说明。

如果未来引入弹弓带质量、效率随弹丸质量变化或释放过程耦合，固定弹簧才会成为不同于固定动能的更复杂模型；本集不引入这些效应。

## 开场钩子

使用上下分屏展示同一对球：

- 上半屏标记“同一初速度”，重球飞得更远；
- 下半屏标记“同一发射能量”，结果发生明显变化；
- 在 12 秒处同时冻结；
- 旁白只问：两次实验都说自己公平，哪一次才算公平？

最终胜负必须来自实现后的参数扫描。如果固定能量实验在首组参数下没有产生足够清晰的视觉差异，应调整合理参数或改为展示连续结果，而不能捏造反转。

## 科学问题

### 实验 A：真空、同一初速度

忽略空气阻力时：

```text
a = g
```

质量不出现在运动方程中。同一位置、同一速度出发的轻球和重球轨迹完全相同。

这是建立因果关系的基线：质量本身不会让真空抛体获得不同的重力加速度。

### 实验 B：有空气、同一初速度

外形与尺寸相同意味着 `C_d` 和 `A` 相同：

```text
F_d = 1/2 ρ C_d A v²
a_d = F_d / m
```

两球在同一速度下承受近似相同的阻力，但轻球的阻力减速度更大。使用弹道系数表示：

```text
β = m / (C_d A)
```

这一实验中，增大质量等价于增大 `β`。

### 实验 C：固定发射能量

固定初始动能时：

```text
E_k = 1/2 mv₀²
v₀ = sqrt(2E_k / m)
```

质量越大，初速度越低。在真空且同角度、同高落地时：

```text
R = v₀² sin(2θ) / g
  = 2E_k sin(2θ) / (mg)
```

因此真空基线下，固定能量的轻球射程更远。加入空气后，轻球同时具有更高初速度和更低弹道系数，两个效应竞争，结果需要数值求解。

### 同材质同比例放大

这不是主实验，但适合在后半段作为更一般的参数关系：

```text
m ∝ r³
A ∝ r²
β = m / (C_d A) ∝ r / C_d
```

同材质球体放大时，质量增长快于迎风面积，因此不能把“更大”和“只是更重”视为同一个实验。

## 变量与控制

### 主比较物体

- 形状相同；
- 半径相同；
- 阻力系数相同；
- 质量通过密度变化；
- 不使用体积大小暗示质量，避免把尺寸影响混入主实验。

### 建议质量组

原型可使用 `0.1 kg`、`0.2 kg`、`0.5 kg`、`1.0 kg`。正式组数根据手机画面可辨识度控制在 3 至 5 组。

### 保持不变的量

每个阶段必须在画面上明确写出：

```text
同速：v₀ 固定
同能量：E₀ 固定
同外形：A、C_d 固定
```

不能只用旁白口头声明。

## 视觉设计

### 物体身份

- 每个质量在整集保持固定数据色；
- 同一个物体跨实验不换色；
- 质量通过核心标记、尾迹线宽或小型 `m` 标签表达，不改变球的半径；
- 赢家使用光环和线宽，不把整块面板填成数据色。

### 公平条件切换

- 画面上方只保留一个当前实验印章：`同速` 或 `同能量`；
- 切换条件时，保留同一对物体位置，通过速度箭头或能量条重排完成过渡；
- 不同时展示三张大表格；
- 公式从相同能量条分配到不同质量上自然生成。

### 结果表达

- 单次比赛用于建立直觉；
- 连续质量扫描用于避免只凭两个样本下结论；
- 横轴使用质量或弹道系数，纵轴使用射程；
- 固定速度和固定能量的曲线在同一坐标系中切换或叠加，但必须清晰标注控制条件。

## 声音设计

- 真空同速实验两球使用完全重合的释放和落地声；
- 加入空气后，轻球的风声变化更明显，但响度不直接代表真实声压；
- 条件从同速切到同能量时，用能量条重新分配的机械卡扣声；
- 轨迹分离使用连续声像或音高差，不为每个质量叠加独立音效；
- 结论不使用“重球胜利”式庆祝声，因为本集没有永恒赢家。

## 模拟与数据设计

### 复用 T001 求解器

使用相同二次阻力、固定步长积分和落地插值。新增的核心不是另一套运动模型，而是实验条件生成器：

```text
fixed_speed(mass) -> v₀
fixed_energy(mass) -> sqrt(2E₀/mass)
fixed_spring(mass) -> sqrt(ηkx²/mass)
```

在效率 `η` 固定时，后两者应产生等价结果；这个等价性应成为自动测试。

### 参数扫描

后台至少扫描：

```text
mass
ballistic_coefficient
launch_rule
```

可选二维扫描：质量与半径。二维结果用于研究和选取代表参数，正片只在确实有助于理解时展示。

### RunRecord 补充元数据

```text
launch_rule
controlled_quantities
launch_speed_mps
initial_kinetic_energy_j
mass_kg
cross_section_area_m2
ballistic_coefficient_kg_m2
```

## 科学验证

- 关闭阻力、固定初速度时，所有质量轨迹应在容差内重合；
- 固定能量生成的初速度应满足 `1/2 mv² = E₀`；
- 固定弹簧与固定能量在 `E₀ = ηkx²/2` 时应一致；
- 同外形同速条件下，质量增加不应导致初始阻力大小变化，但阻力加速度应下降；
- 时间步减半后，所有代表质量的射程应收敛；
- 文案不得把相关性表述成不带条件的普遍结论。

## 制作验收

- 静音观看时能分辨当前是“同速”还是“同能量”；
- 不使用大小不同的球代表主实验中的质量差；
- 正片明确说出理想固定弹簧与固定能量的等价关系；
- 所有赢家由模拟结果产生，不由 Episode 文案硬编码；
- 结尾能补全问题：`在____保持不变时，更重是否飞得更远？`
- 360 像素宽观看时，两条主要轨迹和控制条件仍可辨认。

## 需要新增的系统能力

- 通用 `launch_rule` 参数生成器；
- 控制变量声明与画面印章；
- 同一实体跨阶段保持视觉身份；
- 质量/弹道系数连续扫描；
- 同一坐标系切换多组控制条件；
- 自动检查数学等价实验，避免制作重复或伪独立结论。

## 暂不包含

- 弹弓带自身质量与释放动力学；
- 效率随质量变化；
- 非球形物体姿态变化；
- 尺寸变化导致的 `C_d` 随雷诺数变化；
- 真实材料密度数据库；
- 把“重物抗风”扩展成弹道学完整课程。
