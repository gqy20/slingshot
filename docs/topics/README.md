# 物理可视化选题库

这里收集面向理工科大学生、研究生、工程师与科学爱好者的物理可视化选题。

内容不按高中教材章节组织，而是从一个可观察、可下注、可模拟的问题出发。数学门槛尽量控制在高中范围内，同时通过模型假设、参数扫描、数值方法或工程边界，为已经学过大学物理的观众提供第二层价值。

## 选题原则

一个适合本系列的选题应尽量满足以下条件：

- 前 15 秒能提出一个结果不明显的问题；
- 可以先给出直觉预测，再用确定性模拟验证；
- 至少有一个值得改变的参数或模型假设；
- 结果能通过轨迹、矢量、场、曲线或能量流直观表达；
- 能明确区分“模型算出的结果”和“现实世界的全部细节”；
- 单集围绕一个核心转折，不追求讲完整章课程。

推荐的五分钟结构：

```text
0:00–0:15  现象与下注
0:15–0:50  最简单的直觉或理想模型
0:50–2:00  动画实验验证
2:00–3:40  改变一个现实条件
3:40–4:30  参数关系或更一般的规律
4:30–5:00  模型边界与下一个问题
```

## T001 · 45° 真的总是飞得最远吗？

**开场问题**

把五个完全相同的球以不同角度发射，45° 会赢；加入空气后，它还会赢吗？

**核心物理**

- 真空、同一初速度、同一发射与落地高度时，最佳角度为 45°；
- 使用二次阻力 `F_d = 1/2 ρ C_d A v²`；
- 用数值扫描而不是预设答案寻找最大射程；
- 用弹道系数 `β = m / (C_d A)` 归纳质量、尺寸和阻力系数的共同作用。

**画面设计**

- 先复用现有五角度同步发射；
- 真空轨迹保留为细虚线，空气阻力轨迹逐渐偏离；
- 绘制“发射角—射程”曲线，最大值光标从 45° 向左移动；
- 最后展示最佳角度随弹道系数变化的参数图。

**科学边界**

首版使用常数阻力系数，不声称自动模拟物体周围的真实湍流。

## T002 · 更重的物体真的飞得更远吗？

**开场问题**

外形相同的轻球和重球同时发射，谁先落地、谁飞得更远？

**核心物理**

依次比较三层实验：

1. 真空中固定初速度；
2. 空气中固定初速度；
3. 空气中固定发射能量。

在理想效率固定时，“固定初始动能”和“同一弹簧、相同拉伸距离”是同一类控制条件，不应作为两套独立实验。在固定初速度下，较大的 `m / (C_d A)` 通常意味着相对更弱的减速；固定发射能量时，质量还会改变初速度，因此答案可能反转。

**画面设计**

- 三个实验室并排切换，只改变顶部的控制条件；
- 用同一颜色表示同一质量，避免跨实验颜色语义变化；
- 同时显示初速度、弹道系数和最终射程；
- 结果不是宣布一个赢家，而是展示“问题必须先说清保持什么不变”。

**科学边界**

必须区分“同外形不同密度”和“同材质同比例放大”，后者满足质量随尺寸立方增长、迎风面积随平方增长。

## T003 · 碰撞力为什么不是一个数？

**开场问题**

同一颗球以相同速度撞墙，硬墙和软垫给它的冲量可以相同，为什么破坏程度却完全不同？

**核心物理**

- `J = Δp = ∫F dt`；
- 相同曲线面积可以对应不同峰值和不同接触时间；
- `F_avg = J / Δt` 只是平均量，不是瞬时峰值；
- 引入弹簧—阻尼接触模型，比较刚度和阻尼变化。

**画面设计**

- 左侧播放慢动作碰撞，右侧实时绘制力—时间曲线；
- 用面积填充表示冲量；
- 将窄而高、宽而低的两条曲线叠加；
- 明确标出采样频率对观测峰值的影响。

**科学边界**

Godot 离散碰撞产生的单步冲量不能直接冒充材料的真实瞬时接触力。正式结果应来自声明清楚的接触模型或预设力脉冲。

## T004 · 最短的路为什么不一定最快？

**开场问题**

两个小球同时从同一点出发：一个走直线，一个先快速俯冲再绕远，谁先到终点？

**核心物理**

- 重力势能转化为动能；
- 路径更陡可以更早获得速度；
- 比较直线、圆弧和摆线；
- 最速降线是摆线，而不是几何最短路径。

**画面设计**

- 三条轨道同步释放小球；
- 轨道下方实时展开速度条和累计时间；
- 用路径长度与到达时间两列结果制造对照；
- 结尾从实验轨迹过渡到摆线生成动画。

**科学边界**

首版忽略滚动转动惯量和摩擦，物体视为沿轨道滑动的质点；可以在结尾说明真实滚动物体会改变时间，但不会把“最短等于最快”变成真命题。

## T005 · 下落的弹簧底部为什么会悬空？

**开场问题**

一根竖直悬挂、已经被拉长的弹簧松手后，为什么顶部开始下落，底部却会短暂停在原处？

**核心物理**

- 弹簧张力变化以有限速度沿弹簧传播；
- 松手前底部受到重力和张力平衡；
- 用质点—弹簧链观察压缩波向下传播；
- 比较弹簧刚度、线密度与传播速度。

**画面设计**

- 左侧正常播放，右侧把每一段的张力画成颜色带；
- 用波前标记“支撑已经撤除”的信息传播位置；
- 绘制顶部、中部、底部三个质点的位置—时间曲线；
- 最后改变弹簧刚度，比较悬停持续时间。

**科学边界**

离散质点数量会影响波速和色散，需要做网格收敛检查，并说明模型弹簧与真实 slinky 的复杂接触不同。

## T006 · 轨道运动为什么是永远落不到地面的自由落体？

**开场问题**

从高处水平发射一个物体，不断增加初速度，它何时不再落地，而是开始绕地球运行？

**核心物理**

- 万有引力随距离平方反比变化；
- 地表在物体下落时也持续弯曲；
- 从落地轨迹连续过渡到圆轨道、椭圆轨道和逃逸轨迹；
- 比较第一宇宙速度与逃逸速度，但不把大气层忽略掉的模型当作现实发射方案。

**画面设计**

- 固定发射点，逐次增加水平速度；
- 将地球曲率从近似平地逐渐拉远成完整圆形；
- 速度滑杆与轨道类型同步变化；
- 显示动能、引力势能和总机械能的关系。

**科学边界**

基础模型采用球对称地球、无大气、无自转；随后明确说明真实低轨飞行仍受稀薄大气阻力影响。

## T007 · 欧拉法为什么能让摆越荡越高？

**开场问题**

一个没有外力、没有阻尼的理想摆，交给计算机模拟后，为什么可能越荡越高甚至绕圈？

**核心物理**

- 连续微分方程必须被离散成时间步；
- 显式欧拉、半隐式欧拉和四阶 Runge–Kutta 的误差不同；
- 比较轨迹误差与总能量漂移；
- 时间步减半时，观察结果是否收敛。

**画面设计**

- 三个完全相同的摆并排运行不同积分器；
- 下方绘制总能量随时间的曲线；
- 将时间步放大，让误差快速显现；
- 最后恢复小时间步，展示精度与计算成本的交换。

**科学边界**

重点不是宣布某个算法永远最好，而是说明算法、步长和问题性质共同决定误差。

## T008 · 两个摆为什么会轮流停止？

**开场问题**

只推动左边的摆，为什么过一会儿它几乎停下，而右边的摆开始大幅摆动，随后又交换回来？

**核心物理**

- 两个弱耦合振子的能量交换；
- 对称模和反对称模叠加形成拍频；
- 耦合强度决定交换周期；
- 轻微失谐会阻止完整能量转移。

**画面设计**

- 两个摆通过可见的弱弹簧或公共支架连接；
- 用流动光点表示能量从一个振子转移到另一个；
- 同时显示两个振幅包络线；
- 改变一个摆的长度，展示失谐后的不完整交换。

**科学边界**

首版使用小角度近似解释简正模，同时允许数值模拟使用完整的 `sin θ` 方程展示近似何时失效。

## T009 · 加一块偏振片，为什么光反而通过了？

**开场问题**

两块互相垂直的偏振片会挡住光；在它们中间加入第三块偏振片，为什么屏幕反而重新亮起来？

**核心物理**

- 偏振方向可以表示为二维矢量；
- 每块偏振片都对电场方向做一次投影；
- 使用马吕斯定律 `I = I₀ cos²θ`；
- 中间偏振片把一次 90° 投影拆成两次较小角度投影。

**画面设计**

- 同时展示光强、偏振矢量和三块偏振片方向；
- 先旋转第二块使画面变暗，再插入 45° 偏振片；
- 用矢量投影动画解释强度变化；
- 扫描中间偏振片角度，绘制透射强度曲线。

**科学边界**

首版讨论理想线偏振片和单色光，不展开真实偏振片的吸收、波长依赖和消光比。

## T010 · 磁铁为什么会在铜管里慢慢落下？

**开场问题**

铜不吸磁铁，为什么磁铁穿过铜管时却像落进了黏稠液体？

**核心物理**

- 运动磁铁改变穿过导体回路的磁通量；
- 感应电流产生反抗磁通变化的磁场；
- 电磁阻力在一定范围内可近似与速度有关；
- 重力与电磁阻力平衡后出现近似终端速度；
- 机械能最终主要转化为焦耳热。

**画面设计**

- 将铜管切成一系列可视化导体环；
- 用颜色和箭头表示磁通变化、感应电流与反向磁场；
- 并排比较塑料管、完整铜管和开槽铜管；
- 绘制速度与能量流向，揭示“没有接触”仍能耗散机械能。

**科学边界**

首版可以使用经验证的等效阻尼或离散回路模型，不声称完成真实三维涡流有限元计算。若需要精确对应实验，应使用测量数据或外部电磁求解器校准参数。
